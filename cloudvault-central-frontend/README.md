# CloudVault Central - Frontend

This is a Vite + React + Tailwind frontend for the CloudVault Central demo.

Install and run:

```powershell
cd cloudvault-central-frontend
npm install
npm run dev
```

The dev server runs on http://localhost:3000 and proxies API calls to the backend by using relative `/api` paths. Ensure the backend is running at http://localhost:8000.
